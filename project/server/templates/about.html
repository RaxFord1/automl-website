About RaxFord - AutoML


My name is Dima Dzundza, the creator of RaxFord - AutoML.

Here, you can register and log in, upload your dataset and train your neural network with size of your choice.

After training you can see result of your runs and download checkpoint.

RaxFord - AutoML also includes an admin panel where you can see the machine load.

The frontend of this application is built with Flask's templates (ninja2). You can find the source code { here }.

The backend is powered by Python, Flask, and SQLAlchemy. The source code for the backend is available { here }.

PostgreSQL is used in order to manage account and session storage.

The application is platform-agnostic and may be hosted in any environment. It is released under an open-source license, which grants you the freedom to leverage the project for commercial purposes.
